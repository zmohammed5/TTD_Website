# Tiny Todd's Daycare Inc. - Website

A responsive, mobile-first website developed for Tiny Todd's Daycare Inc., a licensed childcare provider in Richmond Hill, Queens, NY.

## Project Overview

This website was built as a freelance project for a local daycare business with 20+ years of operation. The goal was to create a professional online presence that effectively communicates the daycare's services, facilitates parent inquiries, and improves local search visibility.

**Live Site:** [zmohammed5.github.io/TTD_Website](https://zmohammed5.github.io/TTD_Website/)

## Features

- **Responsive Design** - Mobile-first approach ensuring optimal viewing across all devices
- **Multi-page Architecture** - Organized content across Home, About, Programs, Gallery, Testimonials, Enrollment, and Contact pages
- **Interactive Forms** - Contact and enrollment inquiry forms with client-side validation
- **Google Maps Integration** - Embedded maps and direct links to business location
- **SEO Optimization** - Structured data (JSON-LD), Open Graph meta tags, and semantic HTML
- **Accessibility** - ARIA labels, semantic markup, and keyboard navigation support
- **Performance** - Optimized CSS, minimal JavaScript, and efficient asset loading

## Tech Stack

- **HTML5** - Semantic markup and structured content
- **CSS3** - Custom properties (variables), Flexbox, CSS Grid, animations
- **JavaScript** - Form validation, mobile navigation, scroll interactions
- **Google Fonts** - Fredoka One & Nunito typefaces
- **GitHub Pages** - Static site hosting

## Key Implementation Details

### Responsive Design
- CSS Grid for page layouts with flexible column configurations
- Flexbox for component-level alignment
- Mobile hamburger navigation with smooth transitions
- Breakpoints optimized for common device sizes

### SEO & Discoverability
- JSON-LD structured data for local business schema
- Open Graph tags for social media sharing
- Semantic HTML5 elements (header, nav, main, section, footer)
- Descriptive meta tags and title hierarchy

### User Experience
- Back-to-top button for long pages
- Smooth scroll behavior
- Form validation with user feedback
- Clickable contact information (tel:, mailto:, maps)
